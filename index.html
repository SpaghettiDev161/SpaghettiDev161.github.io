External Scripts Used:

  • Fabric.js: A powerful canvas library used for drawing and manipulating images.
  • QRCode.js: A JavaScript library for generating QR codes.
  • Tailwind CSS: A utility-first CSS framework used for styling.

Data Connected:

  • User-uploaded images
  • Text data from the inputs (e.g., Name, Quote, Place, etc.)
  • URL data provided by the user for citing sources

Note: External libraries are used for rendering the canvas and generating QR codes, while input data is processed locally on the user's browser. No personal data is sent to external servers.
